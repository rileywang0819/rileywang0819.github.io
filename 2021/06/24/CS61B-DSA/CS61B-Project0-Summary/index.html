Overview

This is the first project of CS61B, 2018sp version.

The goal for this project is to write a program simulating the motion of N objects in a plane, accounting for the gravitational forces mutually affecting each object as demonstrated by Sir Isaac Newtonâ€™s Law of Universal Gravitation. Ultimately, a program NBody.java that draws an animation of bodies floating around in space tugging on each other with the power of gravity will be created.

All prerequisite of this assignment is basic Java knowledge. Since this is an easy project with a great deal of scaffolding, I recommend using text editor, such as VS Code, to finish it. But it is fine if you prefer to use IDE.
